Download

You need to download both the server and the agent in order to use GoCD. Both are free.

Scripting GoCD installations gets easier now. Both GoCD Server and Agent are now available on public YUM and APT repositories!

If you need help downloading, installing or using GoCD please join the GoCD Users Google Group. ThoughtWorks also provides commercial help for GoCD. They can be reached at support@thoughtworks.com.
