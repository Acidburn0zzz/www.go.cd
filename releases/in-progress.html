15.2 (beta)

Improvements to the console log

Not only does go now support rendering of ansi color codes to show you much more beautiful console logs. Much more screen real-estate is used to show you what you need to see. Additionally, go will automatically follow the logs as your build produces it, very much like your favorite terminal program.

Authentication end-point

Starting v15.2 Go Server will expose authentication end-point. What this means is Go users can add "custom" authentication schemes through plugins.

Setup hostname when auto-registering agents

You can now specify a property agent.auto.register.hostname to setup the hostname when auto-registering an agent.

API Improvements

Go's APIs are fairly old, have inconsistent and unpredictable content types (csv, xml, json, plain text).

Going forward, we would like to announce an ongoing effort to improve and migrate these APIs to use something that is more modern, easy to discover, learn and build API clients for.

Learn more about the new API on our api documentation site.
